# SGCO
SGCO (Sistema de Gestão para Consultórios Odontológicos) é uma plataforma de apoio a odontólogos e demais profissionais envolvidos em rotinas de gerenciamento de pacientes e consultórios. Foi idealizado como projeto final para a disciplina de Análise e Projeto de Sistemas no curso de Ciência da Computação na UFCA.
