# Sistema de Gestão para Consultórios Odontológicos (SGCO)

### Visão Geral

O SGCO é uma plataforma de apoio a odontólogos e demais profissionais envolvidos em ambientes de consultórios. Sua estrutura divide-se nos seguintes componentes:

- **Gerenciador de Pacientes**: módulo especializado no armazenamento de informações de pacientes, como dados pessoais, históricos de medicamentos, consultas e procedimentos. 
  
- **Gerador de documentos**: módulo especializado na geração de documentos pertinentes à rotina desses profissionais, como encaminhamentos e receitas.
  

### Histórias de Usuário

- **Cadastro de Pacientes**: Eu, como dentista, quero cadastrar dados pessoais (nome, CPF, data de nascimento, contato e anamnese básica) para identificar e gerenciar meus pacientes.
  
- **Linha do Tempo do Paciente**: Eu, como dentista, quero registrar cada procedimento realizado (ex: limpeza, restauração) com data, forma de pagamento e observações, para manter o histórico clínico dos pacientes atualizado.
  
- **Registro de Histórico Medicamentoso**: Eu, como dentista, quero listar alergias e remédios de uso contínuo no perfil do paciente para evitar prescrições perigosas.
  
- **Repositório de Receituários**: Eu, como dentista, quero visualizar todos os receituários já emitidos para um paciente específico para acompanhar a evolução do tratamento.
  
- **Acervo de Procedimentos**: Eu, como dentista, quero visualizar um histórico detalhado de atendimentos (paciente, horário, procedimentos, pagamento e anotações) realizados em minha clínica em um determinado período de tempo, para obter dados sobre meus atendimentos e caixa.
  

- **Gestão de Templates**: Eu, como administrador, quero criar modelos padrão de atestados e encaminhamentos com campos dinâmicos para automatizar a escrita.
  
- **Emissão de Atestado Rápido**: Eu, como dentista, quero selecionar um paciente e um motivo (repouso, consulta) e gerar um PDF de atestado já preenchido e pronto para imprimir/assinar para agilizar a emissão de atestados.
  
- **Prescrição Inteligente (Receituário)**: Eu, como dentista, quero selecionar medicamentos de uma lista pré-definida (posologia padrão) para gerar uma receita em segundos.
  
- **Encaminhamento para Especialistas**: Eu, como dentista, quero gerar documentos de encaminhamento para especialistas (ex: Radiologista, Ortodontista) contendo automaticamente os dados clínicos relevantes do prontuário para agilizar a emissão de encaminhamentos.
  
- **Acervo de Documentos**: Eu, como dentista, quero visualizar todos os documentos já emitidos através do sistema para fins de gerenciamento.
